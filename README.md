# oxygen.go

A modern blog web application currently under development.

## About

Oxygen.go is a lightweight and fast blog platform built with:
- [Go](https://golang.org/) - Backend programming language
- [Fiber](https://gofiber.io/) - Web framework
- [Pug](https://pugjs.org/) - Template engine
